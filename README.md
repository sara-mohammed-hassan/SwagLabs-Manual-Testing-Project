# SwagLabs-Manual-Testing-Project
## Project Overview

This project demonstrates manual testing activities performed on the Swag Labs web application.

The objective of this project is to validate the functionality, usability, and reliability of the application through structured test cases and defect reporting.

## Application Under Test

Swag Labs Website

https://www.saucedemo.com/

## Testing Scope

The following modules were tested:

* Login
* Inventory / Products
* Product Details
* Shopping Cart
* Checkout
* Logout

## Testing Activities

* Requirement Analysis
* Test Scenario Creation
* Test Case Design
* Test Execution
* Defect Reporting
* Defect Tracking
* Test Documentation

## Tools Used

* Jira
* Xray
* Microsoft Excel
* GitHub

## Deliverables

* Test Cases Document
* Bug Reports Document
* Jira Test Cases
* Jira Defects
* Test Evidence Screenshots

## Project Outcome

The testing process successfully identified and documented defects while ensuring the core functionalities of the application were validated through manual testing techniques.

## Traceability

Test Cases were managed using Jira Xray.
Defects were linked to related test cases.
Test execution results were tracked through Jira.
